ArtPro - MINI SITE E-COMMERCE

1. DESCRIPTION DU SITE:

ArtStore est un mini site e-commerce dédié à la vente de materiel de dessin.
Le site permet :
* La consultation des produits artistiques.
* La recherche de produits.
* Le filtrage des produits par catégorie.
* L'inscription d'un utilisateur.
* La connexion d'un utilisateur.
* La saisie d'une commande avec formulaire de livraison.

Les données sont gérées côté client grâce à JavaScript et localStorage, sans utilisation d'un backend.


2. INSTRUCTIONS D'UTILISATION:

2.1 Ouvrir le fichier index.html.
2.2 Accéder à la page Produits.
2.3 Parcourir les catégories ou utiliser la barre de recherche(recherche par categorie seulement).
2.4 Accéder à la page Commande.
2.5 Remplir le formulaire de livraison.
2.6 Se connecter pour pouvoir finaliser la commande.
2.7 Un message de confirmation ou d'erreur s'affiche selon l'état de connexion de l'utilisateur.

-Fonctionnalités non finalisées :
    -gestion compléte du panier d'achat.
    -affichage automatique des produits ajoutés dans la page de commande.
    -calcul dynamique du montant total de la commande.



3. TECHNOLOGIES UTILISÉES

* HTML5
* CSS3
* JavaScript
* LocalStorage
